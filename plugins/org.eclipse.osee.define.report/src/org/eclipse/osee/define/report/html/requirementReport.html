Megumi Telles Requirement Report Generates a Requirement Report
  • Selected Branch - provide the Branch to obtain the Requirement Report for
  • SRS Root Directory - provide the requirement folder to recurse into for files
  • Start Button - generates the report


function oseeAppStart(oseeAppParams) { var branch = oseeAppParams["selected_branch"]; var rootDir = oseeAppParams["srs_root_dir"]; window.location.assign("../requirements?branch="+branch+"&srs_root="+rootDir); }
